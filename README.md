# Trabajo Pagina Web Analisis de Felicidad
esta pagina web es un proyecto de la Universidad Tecnologica de Bolivar en el cual se buscaba hacer una pagina web en la que se hiciera uso de graficas segun los datos de la pagina web world happiness report
este trabajo fue diseñado en el primer corte del primer semestre del programa de ciencias en datso perteneciente a la escuela de transformacion digital  Junto con el Profesor: David Sierra Porta 
En el trabajo primero se descargaron las bases de datos de la pagina world happiness report luego se subieron a google colab para poder crear las graficas y posteriormente descargarlas para poder usarlas en github. para crear nuestra pagina web para el diseño de la pagina web nos apoyamos en la inteligencia artificial gemini y algunos detalles que se ajustaron manualmente como la posicion de cada grafica o de los cuadros ademas del tamaño de esta, yu para la explicacion de cada grafica se hizo uso de la inteligencia Artificial Chatgpt sin embarco tambien se hicieron correciones manuales sobre todo el la forma en que se explicaba cada grafica. este programa presenta una estructura basada 100% en html
